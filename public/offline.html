You're Offline

No connection to your local AI model

StudyBuddy works offline — but it needs to reach your local AI model server the first time you open it. After that, it can work without internet.

To reconnect:

  1. Connect to WiFi: Make sure your phone/device is on the same network as your computer running the AI model.
  2. Check the server: Ensure the StudyBuddy server is running on your computer (you should see the server logs).
  3. Tap the button below to retry. If you're connected, StudyBuddy will load and cache itself.
💡 How it works: When you first open StudyBuddy, it caches all the code, images, and offline pages on your device. Then, when your connection drops, it can show you the cached pages instantly. API calls (like asking a question) will fail gracefully if you're truly offline.
